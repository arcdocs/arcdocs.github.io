Running Chemkin (ARC4 only)Â¶

ANSYS Chemkin provides all the tools for fast, accurate combustion and reaction flow simulations.

Developer WebsiteÂ¶

More information about Chemkin can be found on the Ansys Website.

Getting startedÂ¶

To use Chemkin, you must have connected to ARC using X graphical forwarding, loaded ansys/19.2 and assigned licence server information.

Note

More details about using Ansys for the first time and setting license server details can be found on the Ansys page.

Then run the run_chemkin.sh script passing it the argument Pro to start up Chemkin.

$ module load ansys

$ run_chemkin.sh Pro
